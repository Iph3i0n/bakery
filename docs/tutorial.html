Installation This project can be loaded in a couple of ways CDN (Recommended) This will allow you to use all components with ease in any project if you include this in the head of the HTML. It can be included after the body but this can result in flashes on load, as unknown components will be rendered as if they were Forms All forms are compatible with the standard HTML form element. The provide added functionality around validation and styling. They are included in the f- namespace and must be withing a form element to function.
